- 소스코드 ZIP = 오브젝트 파일 ❌
- TypeScript 파일 = 제출 대상 ❌

### ✅ 실제로 필요한 것
**한국저작권위원회의 "오브젝트 파일"**은:
- 👉 **프로그램이 실제로 어떻게 동작하는지 확인 가능한 형태**
- 👉 웹 기반 앱에서는 **기능 흐름 캡처 이미지 = 정식 복제물**
- 👉 실행 파일(.exe, .apk)이 없으므로 **화면 캡처가 대안**

## 🖼️ 화면 캡처 준비 가이드

### 필요한 캡처 화면

#### 1️⃣ 그림 만들기 모듈 (10-15장)

**필수 화면**:
1. 시작 화면 (그림 만들기 선택)
2. **음성 입력 화면** (말로 설명하기)
3. 음성 인식 중 화면
4. 프롬프트 입력 화면
5. **손글씨 입력 화면** (직접 입력)
6. 손글씨 인식 결과
7. **사진 업로드 화면**
8. AI 이미지 생성 중 (로딩)
9. **생성된 이미지 결과 화면** ⭐
10. 다운로드/공유 화면
11. QR 코드 공유 화면
12. 내 작품 목록

**캡처 방법**:
```
브라우저에서:
1. https://story-maker-4l6.pages.dev 접속
2. 그림 만들기 기능 실행
3. 각 단계마다 스크린샷 (F12 또는 캡처 도구)
4. 전체 화면 + UI 요소 모두 포함
```

---

#### 2️⃣ 글쓰기 모듈 (10-15장)

**필수 화면**:
1. 시작 화면 (글쓰기 선택)
2. **장르 선택 화면** (일기, 편지, 시 등)
3. 빈 편집기 화면
4. **텍스트 입력 화면**
5. **AI 도우미 메뉴** (이어쓰기, 문법교정 등)
6. AI 이어쓰기 결과
7. 문법 교정 결과
8. 감정 강화 결과
9. **완성된 글 화면** ⭐
10. 이미지 생성 (글 기반)
11. 저장/공유 화면
12. 내 작품 상세 화면

---

#### 3️⃣ 동화책 만들기 모듈 (10-15장 + PDF)

**필수 화면**:
1. 시작 화면 (동화책 만들기 선택)
2. **주제 입력 화면**
3. AI 초안 생성 중
4. **생성된 3페이지 초안** ⭐
5. 편집기 화면 (페이지 목록)
6. 텍스트 수정 화면
7. **삽화 생성 화면** ⭐
8. 생성된 삽화 결과
9. 페이지 추가 화면
10. **전체 동화책 미리보기**
11. **PDF 내보내기 설정** (세로/가로 선택)
12. 생성된 PDF 미리보기
13. 저장/공유 화면

**추가 제출**:
- ✅ **실제 생성된 PDF 파일 1-2개** (샘플)

## 📝 제출 서류 작성 가이드

### 기능 설명서 (각 모듈별)

**예시: 1_그림만들기/기능설명.txt**
```
[그림 만들기 모듈]

1. 주요 기능
   - 음성 입력으로 그림 설명
   - 손글씨 인식 (OpenAI Vision)
   - 사진 참고 업로드
   - AI 이미지 생성 (DALL-E 3)

2. 화면 흐름
   01_시작화면: 그림 만들기 선택
   02_음성입력: 마이크 버튼 클릭하여 음성 입력
   03_음성인식중: Web Speech API로 음성 텍스트 변환
   04_프롬프트입력: 변환된 텍스트 확인 및 수정
   05_손글씨입력: 손으로 쓴 글 사진 찍기
   06_손글씨인식: AI가 손글씨를 텍스트로 인식
   07_사진업로드: 참고할 이미지 업로드
   08_AI생성중: DALL-E 3로 이미지 생성 (약 10-15초)
   09_생성결과: 완성된 이미지 표시
   10_다운로드공유: 이미지 저장 또는 QR 코드 공유

3. 기술적 특징
   - Web Speech API (음성 인식)
   - OpenAI Vision API (손글씨 인식)
   - OpenAI DALL-E 3 API (이미지 생성)
   - IndexedDB (로컬 저장)

4. 사용자 경험
   - 노인 친화적 UI (큰 버튼, 명확한 텍스트)
   - 음성 입력으로 쉬운 접근성
   - 결과물 즉시 확인 및 다운로드
```

```bash
# 방법 1: 직접 캡처

```

### 2단계: PDF 샘플 생성

```bash
# 동화책 만들기로 실제 PDF 2-3개 생성
1. 동화책 만들기 접속
2. 주제 입력: "우주를 여행하는 고양이"
3. 초안 생성 → 편집 → 삽화 생성
4. PDF 내보내기 (세로/가로 각 1개)
5. 파일 다운로드: 샘플_동화책1.pdf
```
