## 🎯 목표
각 모듈별로 10-15장의 화면 캡처 수집

## 🖥️ 캡처 준비

### 필수 도구
- **프로덕션 URL**: https://story-maker-4l6.pages.dev
- **캡처 도구**: 
  - Windows: `Win + Shift + S` (캡처 도구)
  - Mac: `Cmd + Shift + 4`
  - Linux: `PrtScn` 또는 `Gnome Screenshot`
  - 브라우저: F12 → 스크린샷 기능

```

...

```

## 📋 1. 그림 만들기 모듈 (10-15장)

### 캡처 순서

#### 01. 시작 화면
```
URL: https://story-maker-4l6.pages.dev
화면: Welcome 페이지
- "HI-DI Edu" 로고
- "스토리 메이커" 제목
- "시작하기" 버튼
```

#### 02. 홈 화면
```
동작: "시작하기" 버튼 클릭
화면: 3가지 옵션
- 그림 만들기
- 글쓰기  
- 동화책 만들기
```

#### 03. 그림 만들기 선택
```
동작: "그림 만들기" 클릭
화면: 2가지 입력 방법
- 말로 설명하기 (마이크 아이콘)
- 직접 입력하기 (키보드 아이콘)
```

#### 04. 말로 설명하기 화면
```
동작: "말로 설명하기" 클릭
화면: 음성 입력 화면
- 마이크 버튼 (빨간색)
- "마이크를 클릭해서 말씀해주세요" 안내
```

#### 05. 음성 인식 중 (선택사항)
```
동작: 마이크 버튼 클릭
화면: 음성 인식 중 표시
- 파형 애니메이션 또는 "듣는 중..." 메시지
```

#### 06. 프롬프트 입력 완료
```
화면: 텍스트 입력창에 내용 표시
예: "우주를 여행하는 귀여운 고양이"
```

#### 07. 직접 입력하기 화면
```
동작: 뒤로 가기 → "직접 입력하기" 클릭
화면: 텍스트 입력 화면
- 입력창 (큰 텍스트 영역)
- 손글씨 입력 버튼
- 사진 업로드 버튼
- 스타일 선택 옵션
```

#### 08. 손글씨 입력 버튼
```
화면: 손글씨 입력 기능 강조
- "손글씨로 쓰기" 버튼 확대 캡처
```

#### 09. 사진 업로드 버튼
```
화면: 사진 업로드 기능 강조
- "사진 업로드" 버튼 확대 캡처
```

#### 10. AI 이미지 생성 중 (중요 ⭐)
```
동작: 프롬프트 입력 후 "만들기" 클릭
화면: 로딩 화면
- 로딩 애니메이션
- "AI가 이미지를 만들고 있어요..." 메시지
```

#### 11. 생성된 이미지 결과 (핵심 ⭐⭐⭐)
```
화면: 완성된 이미지 표시
- 생성된 이미지 (전체 화면)
- 다운로드 버튼
- 공유하기 버튼
- 다시 만들기 버튼
```

#### 12. 다운로드/공유 버튼
```
화면: 하단 버튼 강조
- 다운로드 (초록색)
- 공유하기 (파란색)
```

#### 13. QR 코드 공유 (선택사항)
```
동작: "공유하기" 클릭
화면: QR 코드 모달
- QR 코드 이미지
- "QR 코드 저장" 버튼
```

#### 14. 내 작품 목록
```
URL: https://story-maker-4l6.pages.dev/my-works
화면: 저장된 그림 목록
- 썸네일 이미지들
- 날짜 정보
```

#### 15. 작품 상세 화면
```
동작: 썸네일 클릭
화면: 작품 상세
- 이미지 크게 표시
- 다운로드/수정/삭제 버튼
```

## 📋 2. 글쓰기 모듈 (10-15장)

### 캡처 순서

#### 01. 홈 화면
```
URL: https://story-maker-4l6.pages.dev
동작: "시작하기" → "글쓰기" 클릭
```

#### 02. 글쓰기 옵션
```
화면: 3가지 시작 방법
- 💡 연습하기 (주제 추천)
- 📝 장르 선택하기
- ✍️ 자유롭게 쓰기
```

#### 03. 연습하기 화면
```
동작: "연습하기" 클릭
화면: AI 주제 추천
- 추천 주제 3-4개 표시
- "이 주제로 쓰기" 버튼
```

#### 04. 장르 선택 화면
```
동작: 뒤로 가기 → "장르 선택하기" 클릭
화면: 6가지 장르
- 일기
- 편지
- 수필
- 시
- 소설
- 자서전
```

#### 05. 일기 편집기 (빈 화면)
```
동작: "일기" 클릭
화면: 빈 편집기
- 제목 입력창
- 본문 입력창 (큰 텍스트 영역)
- AI 도우미 버튼
```

#### 06. 텍스트 입력 중
```
화면: 텍스트 입력된 상태
예:
제목: "행복한 하루"
본문: "오늘은 정말 행복한 하루였다..."
```

#### 07. AI 도우미 메뉴 (중요 ⭐)
```
동작: AI 도우미 버튼 클릭
화면: AI 기능 메뉴
- ✨ 이어쓰기
- 📝 문법 교정
- 💖 감정 강화
- 📚 구성 제안
- ✍️ 문장 다듬기
- 📊 텍스트 분석
- 💡 제목 추천
```

#### 08. 이어쓰기 실행 중 (선택사항)
```
동작: "이어쓰기" 클릭
화면: AI 작업 중
- 로딩 애니메이션
- "AI가 이어쓰고 있어요..." 메시지
```

#### 09. 이어쓰기 결과
```
화면: AI가 추가한 텍스트
- 원본 + AI 생성 텍스트 구분 표시
```

#### 10. 문법 교정 결과
```
동작: "문법 교정" 클릭
화면: 교정된 텍스트
- 수정된 부분 하이라이트
```

#### 11. 완성된 글 (핵심 ⭐⭐⭐)
```
화면: 최종 완성된 글
- 제목 + 본문 전체
- 저장 버튼
```

#### 12. 이미지 생성 (선택사항)
```
동작: "이미지 생성" 버튼 클릭
화면: 글 기반 AI 이미지
```

#### 13. 저장 완료
```
동작: "저장" 버튼 클릭
화면: 저장 완료 메시지
- "작품이 저장되었습니다" 알림
```

#### 14. 내 작품 목록 (글)
```
URL: https://story-maker-4l6.pages.dev/my-works
화면: 저장된 글 목록
- 제목 리스트
- 날짜 정보
```

#### 15. 작품 상세 화면 (글)
```
동작: 제목 클릭
화면: 글 상세 보기
- 전체 내용
- 다운로드/수정/삭제 버튼
```

## 📋 3. 동화책 만들기 모듈 (10-15장 + PDF)

### 캡처 순서

#### 01. 홈 화면
```
URL: https://story-maker-4l6.pages.dev
동작: "시작하기" → "동화책 만들기" 클릭
```

#### 02. 동화책 시작 화면
```
화면: 동화책 만들기 안내
- "어떤 이야기를 만들까요?" 제목
- 주제 입력창
```

#### 03. 주제 입력 화면
```
화면: 주제 입력창 강조
- 입력창에 커서 표시
- 플레이스홀더 텍스트
```

#### 04. 주제 입력 완료
```
화면: 주제 입력된 상태
예: "우주를 여행하는 용감한 토끼 이야기"
```

#### 05. 스타일 선택
```
화면: 동화 스타일 옵션
- 동화 스타일
- 일러스트 스타일
- 수채화 스타일 등
```

#### 06. AI 초안 생성 중 (중요 ⭐)
```
동작: "만들기" 버튼 클릭
화면: 로딩 화면
- 로딩 애니메이션
- "AI가 동화를 만들고 있어요..." 메시지
- 예상 시간: 약 30초
```

#### 07. 생성된 3페이지 초안 (핵심 ⭐⭐⭐)
```
화면: 동화 초안 완성
- 3개 페이지 카드 표시
- 각 페이지 텍스트 일부 보임
- "편집하기" 버튼
```

#### 08. 동화책 편집기
```
동작: "편집하기" 클릭
화면: 편집기 화면
- 왼쪽: 페이지 목록 (3개)
- 오른쪽: 선택된 페이지 내용
```

#### 09. 페이지 텍스트 수정
```
화면: 텍스트 편집 중
- 텍스트 입력창
- 수정된 내용 표시
```

#### 10. 삽화 생성 버튼
```
화면: 삽화 생성 기능 강조
- "삽화 만들기" 버튼
- 각 페이지별 버튼 표시
```

#### 11. 삽화 생성 중 (중요 ⭐)
```
동작: "삽화 만들기" 클릭
화면: 이미지 생성 중
- 로딩 애니메이션
- "AI가 삽화를 만들고 있어요..." 메시지
```

#### 12. 생성된 삽화 (핵심 ⭐⭐⭐)
```
화면: 페이지에 삽화 추가됨
- 이미지 + 텍스트 조합
- 전체 페이지 미리보기
```

#### 13. 전체 동화책 미리보기
```
동작: "미리보기" 버튼 클릭 (있는 경우)
화면: 전체 페이지 슬라이드
- 모든 페이지 순서대로 표시
```

#### 14. PDF 내보내기 설정 (중요 ⭐)
```
동작: "PDF 내보내기" 버튼 클릭
화면: PDF 설정 모달
- 제목 입력
- 저자 입력
- 방향 선택: 세로 / 가로
- 커버 이미지 생성 옵션
```

#### 15. PDF 생성 중
```
화면: PDF 생성 중
- 로딩 바
- "PDF를 만들고 있어요..." 메시지
```

#### ⭐⭐⭐ 16. PDF 다운로드 (필수!)
```
동작: PDF 생성 완료
화면: 다운로드 버튼
- "PDF 다운로드" 버튼 클릭
- 파일 저장 다이얼로그

⚠️ 중요: 실제 PDF 파일 2-3개 저장!
- 샘플1: 세로 방향 PDF
- 샘플2: 가로 방향 PDF
```

## ✅ 캡처 팁

### 화면 크기
- **권장 해상도**: 1920x1080 이상
- **전체 화면 캡처** (브라우저 UI 포함 가능)
- 모바일뷰 아닌 **데스크탑뷰**로 캡처

### 화질
- **PNG 형식** 권장 (JPG보다 선명)
- 압축하지 않은 원본 품질
- 최소 해상도: 1280x720

### 내용
- **UI 요소 모두 포함** (버튼, 메뉴, 텍스트)
- **에러 화면 제외** (정상 작동 화면만)
- **로딩 화면 포함** (AI 작업 중 표시)

### 연속성
- **순서대로 캡처** (01, 02, 03...)
- **기능 흐름이 연결**되도록
- 빠진 화면 없이 완전하게
