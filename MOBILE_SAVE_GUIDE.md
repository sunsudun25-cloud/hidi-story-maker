# 📱 모바일에서 이미지 저장하기

## 💾 저장 위치 안내

### **1. iOS (iPhone/iPad) - Safari**

#### **저장 방법:**
```
1. "💾 저장하기" 버튼 클릭
2. 이미지가 새 탭에서 열림
3. 이미지를 길게 누르기 (2초)
4. 팝업 메뉴에서 "사진에 추가" 선택
5. ✅ 사진 앱에 저장됨!
```

#### **저장 위치:**
```
사진 앱 → 최근 항목 → ai-drawing-xxx.png
```

#### **주의사항:**
- Safari 브라우저는 자동 다운로드를 지원하지 않습니다
- 사용자가 직접 "사진에 추가"를 선택해야 합니다
- Chrome 앱을 사용하면 자동 다운로드 가능

---

### **2. Android - Chrome/Samsung Internet**

#### **저장 방법:**
```
1. "💾 저장하기" 버튼 클릭
2. 자동 다운로드 시작
3. 알림 표시: "다운로드 완료"
4. ✅ Download 폴더에 저장됨!
```

#### **저장 위치:**
```
방법 1: 내 파일 앱 → Download 폴더
방법 2: Chrome → 메뉴 → 다운로드
방법 3: 갤러리 앱 (자동 인덱싱)
```

#### **파일 경로:**
```
/storage/emulated/0/Download/ai-drawing-[timestamp].png
```

```

```

## ❓ 문제 해결

### **Q: iOS에서 "저장하기"를 눌렀는데 아무 일도 안 일어나요**
**A:** 새 탭이 차단되었을 수 있습니다.
```
해결 방법:
Safari 설정 → 팝업 차단 해제
또는
"내 작품 보기" 메뉴 사용
```

### **Q: Android에서 다운로드한 파일을 찾을 수 없어요**
**A:** 다음 위치를 확인하세요:
```
1. 내 파일 앱 → Download 폴더
2. 갤러리 앱 → 최근 항목
3. Chrome → 메뉴 → 다운로드
```

### **Q: 저장된 이미지를 삭제하고 싶어요**
**A:** 
```
방법 1 (브라우저 내):
내 작품 보기 → 이미지 선택 → 🗑️ 버튼

방법 2 (기기):
사진 앱 또는 파일 앱에서 직접 삭제
```

### **Q: "내 작품 보기"에 저장된 이미지가 실제로 어디에 있나요?**
**A:** 
```
IndexedDB (브라우저 내부 저장소)에 저장됩니다.
- 인터넷 연결 없이도 접근 가능
- 브라우저 캐시를 삭제하면 함께 삭제됨
- 실제 파일 시스템에는 저장되지 않음

실제 파일로 저장하려면:
"💾 저장하기" 또는 "📥 다운로드" 버튼 사용
```

## 📊 저장 방식 비교

### **1. "저장하기" 버튼 (실제 파일)**
```
✅ 장점:
- 실제 파일로 저장됨
- 다른 앱에서 사용 가능
- 백업 가능

❌ 단점:
- iOS에서 수동 작업 필요
- 저장 공간 차지
```

### **2. "내 작품 보기" (IndexedDB)**
```
✅ 장점:
- 자동 저장
- 빠른 접근
- 정리 기능 제공

❌ 단점:
- 브라우저 캐시 삭제 시 소실
- 다른 앱에서 접근 불가
- 백업 불가
```

### **권장 사용법:**
```
1. 이미지 생성 → 자동으로 "내 작품 보기"에 저장
2. 마음에 드는 이미지 → "💾 저장하기"로 실제 파일 저장
3. 공유하려면 → "📤 공유하기" 사용
```
