# 🤖 이미지 생성 모델 선택 가이드

## 📍 위치
**그림 만들기 → 연습하기**

## 🎯 사용 방법

### 1️⃣ 모델 선택 드롭다운
페이지 상단에 "🤖 이미지 생성 모델" 드롭다운이 추가되었습니다.

### 2️⃣ 모델 옵션

| 모델 | 설명 | 특징 |
|------|------|------|
| **DALL-E 3** | 기본, 고품질 | ✅ 기본값 (안정) |
| **GPT-Image 1.5** | 차세대 모델 ⚡ | 🆕 최신 고품질 |
| **GPT-Image 1** | 표준 | ⚡ 빠른 생성 |
| **GPT-Image Mini** | 빠른 생성 | ⚡⚡ 가장 빠름 |

### 3️⃣ 자동 폴백
- **GPT-Image 모델 실패 시**: 자동으로 DALL-E 3로 재시도
- **사용자 경험**: 끊김 없는 이미지 생성

---

## 🧪 테스트 방법

### Step 1: 페이지 접속
```
홈 → 그림 만들기 → 연습하기
```

### Step 2: 모델 선택
1. "🤖 이미지 생성 모델" 드롭다운 클릭
2. 원하는 모델 선택 (예: GPT-Image 1.5)
3. 설명 텍스트 확인

### Step 3: 이미지 생성
1. 예시 프롬프트 클릭 또는 직접 입력
2. "🎨 AI에게 그림 만들어달라고 하기" 버튼 클릭
3. 생성 완료 대기

### Step 4: 결과 확인
1. 브라우저 개발자 도구 (F12) → Console 탭 열기
2. 로그 확인:
   ```
   🎨 이미지 생성 시작: {prompt: "...", model: "gpt-image-1.5"}
   🎨 동화 이미지 생성 중: {model: "gpt-image-1.5", ...}
   ✅ 동화 이미지 생성 완료
   ```

## 🚨 문제 해결

### Q1: 드롭다운이 보이지 않아요
**A**: 캐시 문제일 수 있습니다.
- Ctrl+Shift+R (Windows) / Cmd+Shift+R (Mac)로 강력 새로고침
- 최신 배포 URL 사용: https://5c007d93.story-maker-4l6.pages.dev

### Q2: 새 모델이 작동하지 않아요
**A**: 자동 폴백이 작동하여 DALL-E 3로 전환되었을 수 있습니다.
- 개발자 도구 Console에서 로그 확인
- "⚠️ 1차 실패 → dall-e-3로 fallback" 메시지 확인

### Q3: 속도 차이가 느껴지나요?
**A**: 네트워크 상태에 따라 다를 수 있습니다.
- GPT-Image 1.5: 3-5초
- DALL-E 3: 4-6초
- GPT-Image Mini: 2-3초 (예상)
