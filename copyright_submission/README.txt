Story Maker 저작권 등록 제출 자료
===========================================

작성일: 2026-01-07
프로젝트명: Story Maker (AI 스토리 플랫폼)
제출 모듈: 3건

-------------------------------------------
1. 그림 만들기 모듈 (copyright_1_drawing.zip)
-------------------------------------------
- 음성 입력으로 그림 생성
- 손글씨 인식
- 사진 업로드
- AI 이미지 생성 (DALL-E 3)

주요 파일:
- DrawPractice.tsx: 말로 설명하기
- DrawDirect.tsx: 직접 입력
- DrawingResult.tsx: 결과 화면
- imageService.ts: 이미지 관리
- cloudflareImageApi.ts: AI 이미지 생성

-------------------------------------------
2. 글쓰기 모듈 (copyright_2_writing.zip)
-------------------------------------------
- AI 글쓰기 도우미
- 장르별 글쓰기
- 이어쓰기, 문법교정
- 손글씨/음성 입력

주요 파일:
- Write.tsx: 글쓰기 시작
- WriteEditor.tsx: 글쓰기 편집기
- geminiService.ts: AI 도우미

-------------------------------------------
3. 동화책 만들기 모듈 (copyright_3_storybook.zip)
-------------------------------------------
- AI 동화 생성
- 페이지별 삽화
- PDF 내보내기
- 세로/가로 레이아웃

주요 파일:
- Storybook/Create.tsx: 초안 생성
- Storybook/Editor.tsx: 편집기
- Storybook/Export.tsx: PDF 내보내기
- pdfService.ts: PDF 생성
