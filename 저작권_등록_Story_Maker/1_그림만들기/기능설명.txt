───────────────────────────────────────────────────────────────
1. 모듈 개요
───────────────────────────────────────────────────────────────

본 모듈은 노인 사용자를 위한 AI 기반 이미지 생성 도구입니다.
음성 입력, 손글씨 인식, 사진 업로드 등 다양한 입력 방식을 지원하며,
DALL-E 3 및 GPT-Image 모델을 활용하여 고품질 이미지를 생성합니다.

───────────────────────────────────────────────────────────────
2. 주요 기능
───────────────────────────────────────────────────────────────

[2.1] 다양한 입력 방식
  ✓ 말로 설명하기 (음성 인식)
    - Web Speech API 활용
    - 한국어 음성 인식 지원
    - 실시간 텍스트 변환
  
  ✓ 직접 입력하기 (키보드 입력)
    - 텍스트 직접 입력
    - 한글 IME 지원
    - 프롬프트 자동완성
  
  ✓ 손글씨 인식
    - GPT-4o-mini Vision API 활용
    - 손으로 쓴 텍스트 이미지 인식
    - 10-15초 처리 시간
  
  ✓ 사진 업로드
    - 최대 10MB 지원
    - JPEG, PNG, GIF, WebP 형식
    - 드래그 앤 드롭 지원

[2.2] AI 이미지 생성
  ✓ 멀티 모델 지원
    - DALL-E 3 (기본)
    - GPT-image-1.5
    - nano-banana-pro
    - 자동 폴백 시스템
  
  ✓ 다양한 스타일 옵션
    - 동화 스타일
    - 일러스트 스타일
    - 사실적 스타일
    - 수채화 스타일
  
  ✓ 고품질 출력
    - 1024x1024 해상도 (기본)
    - 최대 2K/4K 지원
    - PNG 형식 저장

[2.3] 결과물 관리
  ✓ 다운로드
    - PNG 파일 다운로드
    - 원본 품질 유지
    - 파일명 자동 생성
  
  ✓ 공유하기
    - QR 코드 생성
    - Firebase Storage 업로드
    - 공유 링크 생성
    - QR 이미지 저장
  
  ✓ 다시 만들기
    - 같은 프롬프트로 재생성
    - 다른 스타일 시도
    - 생성 히스토리 저장

[2.4] 내 작품 관리
  ✓ IndexedDB 로컬 저장
    - 브라우저 로컬 데이터베이스
    - 오프라인 접근 가능
    - 용량 제한 없음
  
  ✓ 작품 목록
    - 썸네일 미리보기
    - 날짜별 정렬
    - 카드 레이아웃
  
  ✓ 작품 상세
    - 전체 이미지 보기
    - 프롬프트 확인
    - 생성 정보 표시
  
  ✓ 편집 기능
    - 다운로드
    - 수정 (재생성)
    - 삭제
    - 공유

───────────────────────────────────────────────────────────────
3. 기술적 특징
───────────────────────────────────────────────────────────────

[3.1] 음성 인식 시스템
  - Web Speech API (브라우저 내장)
  - 한국어(ko-KR) 최적화
  - 실시간 스트림 처리
  - 노이즈 필터링

[3.2] 손글씨 인식 (OCR)
  - GPT-4o-mini Vision API
  - 다양한 필체 지원
  - 맥락 기반 인식
  - 고정확도 처리

[3.3] AI 이미지 생성
  - OpenAI DALL-E 3
  - Cloudflare Workers 통합
  - 멀티 모델 폴백
  - 자동 에러 복구

[3.4] 데이터 저장
  - IndexedDB (로컬)
  - Firebase Storage (공유)
  - Base64 이미지 인코딩
  - 메타데이터 관리

4. 사용자 흐름
───────────────────────────────────────────────────────────────

시작 화면
  ↓
홈 화면 (그림 만들기 선택)
  ↓
입력 방식 선택
  ├─ 말로 설명하기 → 음성 인식 → 텍스트 변환
  ├─ 직접 입력하기 → 키보드 입력
  ├─ 손글씨 인식 → 이미지 업로드 → OCR
  └─ 사진 업로드 → 파일 선택
  ↓
프롬프트 확인 및 스타일 선택
  ↓
AI 이미지 생성 (20-30초)
  ↓
결과 이미지 표시
  ↓
작업 선택
  ├─ 다운로드 → PNG 저장
  ├─ 공유하기 → QR 코드 생성
  ├─ 다시 만들기 → 재생성
  └─ 저장 → 내 작품에 추가
  ↓
내 작품 보기
  └─ 작품 목록 → 상세 보기 → 편집/삭제
