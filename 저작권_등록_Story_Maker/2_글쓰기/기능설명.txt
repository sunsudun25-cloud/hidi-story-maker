1. 모듈 개요
───────────────────────────────────────────────────────────────

본 모듈은 노인 사용자를 위한 AI 기반 글쓰기 도우미 프로그램입니다.
주제 추천, 장르별 템플릿, AI 이어쓰기 등 다양한 기능을 통해
누구나 쉽게 글을 작성할 수 있도록 지원합니다.

───────────────────────────────────────────────────────────────
2. 주요 기능
───────────────────────────────────────────────────────────────

[2.1] 3가지 시작 방식
  ✓ 연습하기 (주제 추천)
    - AI가 주제 3-4개 자동 생성
    - 계절, 관심사 기반 추천
    - "이 주제로 쓰기" 원클릭 시작
  
  ✓ 장르 선택하기
    - 일기: 일상 기록 템플릿
    - 편지: 감성적 문구 지원
    - 수필: 자유로운 산문
    - 시: 시적 표현 도움
    - 소설: 스토리 구성 가이드
    - 자서전: 인생 회고 템플릿
  
  ✓ 자유롭게 쓰기
    - 빈 캔버스 제공
    - 제약 없는 자유 작문
    - 나중에 장르 변경 가능

[2.2] AI 글쓰기 도우미 (핵심 기능 ⭐⭐⭐)
  ✓ 이어쓰기
    - 맥락 이해 후 자연스럽게 이어 작성
    - 현재 글 분위기 유지
    - 200-300자 단위 생성
  
  ✓ 문법 교정
    - 맞춤법 검사
    - 띄어쓰기 수정
    - 어색한 표현 개선
    - 수정 부분 하이라이트
  
  ✓ 감정 강화
    - 감정 표현 풍부하게
    - 감성적 어휘 추가
    - 생동감 있는 문장
  
  ✓ 구성 제안
    - 단락 구조 개선
    - 논리적 흐름 조정
    - 서론-본론-결론 가이드
  
  ✓ 문장 다듬기
    - 중복 표현 제거
    - 간결한 문장 제안
    - 가독성 향상
  
  ✓ 텍스트 분석
    - 감정 분석 (긍정/부정/중립)
    - 키워드 추출
    - 읽기 난이도 분석
    - 문장 길이 통계
  
  ✓ 제목 추천
    - 본문 내용 기반
    - 창의적인 제목 3-5개 제안
    - 장르별 맞춤 제목

[2.3] 다양한 입력 방식
  ✓ 키보드 입력
    - 한글 IME 완벽 지원
    - 자동완성 기능
    - 실시간 글자 수 표시
  
  ✓ 손글씨 인식
    - GPT-4o-mini Vision OCR
    - 손으로 쓴 메모 인식
    - 노트 사진 텍스트 변환
  
  ✓ 음성 입력
    - Web Speech API
    - 한국어 음성 인식
    - 구두점 자동 추가

[2.4] 결과물 관리
  ✓ 저장 기능
    - IndexedDB 로컬 저장
    - 자동 저장 (30초마다)
    - 임시 저장 보호
  
  ✓ 다운로드
    - TXT 파일 내보내기
    - DOC 형식 지원
    - UTF-8 인코딩
  
  ✓ 이미지 생성 (선택사항)
    - 글 내용 기반 AI 이미지
    - 썸네일 자동 생성
    - 본문 삽화 추가

[2.5] 내 작품 관리
  ✓ 작품 목록
    - 제목 리스트
    - 날짜별 정렬
    - 장르별 필터
    - 검색 기능
  
  ✓ 작품 상세
    - 전체 내용 보기
    - 메타데이터 표시
    - 편집 이력
  
  ✓ 편집 기능
    - 수정하기
    - 복사하기
    - 삭제하기
    - 공유하기

  ├─ 연습하기 → AI 주제 추천 → 주제 선택

  ├─ 이어쓰기

  ├─ 감정 강화

  ├─ 문장 다듬기
  ├─ 텍스트 분석
  └─ 제목 추천
  ↓

  ↓

  ├─ 저장 → 내 작품에 추가

6. AI 도우미 기능 상세
───────────────────────────────────────────────────────────────

[6.1] 이어쓰기
  입력: 현재까지 작성된 글
  처리: 맥락 분석 → 자연스러운 연결 문장 생성
  출력: 200-300자 이어진 내용
  시간: 5-10초

[6.2] 문법 교정
  입력: 전체 텍스트
  처리: 맞춤법, 띄어쓰기, 문법 검사
  출력: 수정된 텍스트 + 변경사항 하이라이트
  시간: 3-5초

[6.3] 감정 강화
  입력: 감정 표현이 포함된 텍스트
  처리: 감성 어휘 추가, 비유적 표현 삽입
  출력: 더 풍부한 감정이 담긴 텍스트
  시간: 5-10초

[6.4] 구성 제안
  입력: 여러 단락의 긴 글
  처리: 논리적 흐름 분석, 단락 재구성
  출력: 개선된 구조 제안
  시간: 10-15초

[6.5] 문장 다듬기
  입력: 장황하거나 중복된 문장
  처리: 간결화, 중복 제거, 가독성 향상
  출력: 깔끔하게 다듬어진 문장
  시간: 3-5초

[6.6] 텍스트 분석
  입력: 완성된 글
  처리: 감정 분석, 키워드 추출, 통계 계산
  출력: 
    - 감정: 긍정 85%, 부정 5%, 중립 10%
    - 키워드: 행복, 가족, 여행 등
    - 총 글자수: 1,234자
    - 평균 문장 길이: 45자
  시간: 3-5초

[6.7] 제목 추천
  입력: 본문 내용
  처리: 핵심 내용 요약, 창의적 표현 생성
  출력: 3-5개 제목 후보
  시간: 5-10초
