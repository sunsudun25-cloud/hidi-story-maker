📦 제출물 구성
───────────────────────────────────────────────────────────────

본 패키지는 Story Maker의 3개 핵심 모듈에 대한 
저작권 등록 제출물입니다.

1_그림만들기/
  - 화면 캡처 10-15장 (PNG)
  - 기능설명.txt
  - 실제 생성된 이미지 샘플

2_글쓰기/
  - 화면 캡처 10-15장 (PNG)
  - 기능설명.txt
  - 실제 작성된 글 샘플 (TXT/DOC)

3_동화책만들기/
  - 화면 캡처 10-15장 (PNG)
  - 기능설명.txt
  - ⭐⭐⭐ 샘플_동화책_세로.pdf (필수!)
  - ⭐⭐⭐ 샘플_동화책_가로.pdf (필수!)

───────────────────────────────────────────────────────────────
🎯 각 모듈 개요
───────────────────────────────────────────────────────────────

[모듈 1] 그림 만들기
  - AI 이미지 생성 도구
  - 음성/손글씨/사진 입력 지원
  - DALL-E 3 고품질 이미지
  - QR 코드 공유 기능
  - 내 작품 관리

[모듈 2] 글쓰기
  - AI 글쓰기 도우미
  - 장르별 템플릿 (일기, 편지, 수필, 시, 소설, 자서전)
  - AI 이어쓰기, 문법 교정, 감정 강화 등 7가지 기능
  - 텍스트 분석 및 제목 추천
  - 내 작품 관리

[모듈 3] 동화책 만들기
  - AI 동화 스토리 자동 생성
  - 페이지별 삽화 AI 생성
  - PDF 내보내기 (세로/가로 레이아웃)
  - 한글 폰트 지원
  - 내 작품 관리

📸 화면 캡처 가이드
───────────────────────────────────────────────────────────────

[필수 캡처 화면]
  ✓ 시작 화면
  ✓ 홈 화면
  ✓ 기능 선택 화면
  ✓ 입력 화면
  ✓ AI 처리 중 화면 (로딩)
  ✓ 결과 화면 (핵심!)
  ✓ 다운로드/공유 화면
  ✓ 내 작품 목록
  ✓ 작품 상세

[캡처 방법]
  - Windows: Win + Shift + S
  - Mac: Cmd + Shift + 4
  - Linux: PrtScn
  - 브라우저: F12 → 스크린샷

[캡처 품질]
  - 해상도: 1920x1080 이상
  - 형식: PNG (JPG보다 선명)
  - 압축: 사용하지 않음
  - 전체 화면 캡처 (UI 포함)

───────────────────────────────────────────────────────────────
🎨 실제 결과물 샘플
───────────────────────────────────────────────────────────────

[그림 만들기 모듈]
  - AI 생성 이미지 2-3개 (PNG)
  - 예: "우주를 여행하는 고양이" 이미지

[글쓰기 모듈]
  - 완성된 글 1-2개 (TXT 또는 DOC)
  - 예: 일기, 편지, 수필 등
  - 한글 인코딩: UTF-8

[동화책 만들기 모듈] ⭐⭐⭐ 중요!
  - 샘플_동화책_세로.pdf
    → Portrait 방향
    → 그림 위, 글 아래 레이아웃
    → 3-5페이지
    → 한글 텍스트 + AI 이미지

  - 샘플_동화책_가로.pdf
    → Landscape 방향
    → 그림 왼쪽, 글 오른쪽 레이아웃
    → 3-5페이지
    → 한글 텍스트 + AI 이미지
