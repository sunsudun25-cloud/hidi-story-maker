🎯 목표: 3개 모듈별로 10-15장씩 화면 캡처 + 결과물 샘플 수집

───────────────────────────────────────────────────────────────
📱 1단계: 앱 접속
───────────────────────────────────────────────────────────────

URL: https://story-maker-4l6.pages.dev

브라우저 권장:
  - Chrome (최신 버전)
  - Edge (최신 버전)
  - Safari (최신 버전)

───────────────────────────────────────────────────────────────
📸 2단계: 캡처 준비
───────────────────────────────────────────────────────────────

[Windows 사용자]
  - 단축키: Win + Shift + S
  - 또는: Snipping Tool 실행

[Mac 사용자]
  - 단축키: Cmd + Shift + 4

[Linux 사용자]
  - PrtScn 키
  - 또는: gnome-screenshot

저장 위치:
  - 1_그림만들기/ 폴더
  - 2_글쓰기/ 폴더
  - 3_동화책만들기/ 폴더

파일명 형식:
  - 01_시작화면.png
  - 02_홈화면.png
  - 03_기능선택.png
  - ...

───────────────────────────────────────────────────────────────
🎨 3단계: 모듈 1 - 그림 만들기 (15-20분)
───────────────────────────────────────────────────────────────

1. https://story-maker-4l6.pages.dev 접속
   → 캡처: 01_시작화면.png

2. "시작하기" 클릭
   → 캡처: 02_홈화면.png

3. "그림 만들기" 클릭
   → 캡처: 03_입력방식선택.png

4. "말로 설명하기" 클릭
   → 캡처: 04_음성입력화면.png

5. 마이크 클릭 후 "우주를 여행하는 귀여운 고양이" 말하기
   → 캡처: 05_음성인식중.png (선택사항)
   → 캡처: 06_프롬프트입력완료.png

6. 뒤로 가기 → "직접 입력하기" 클릭
   → 캡처: 07_직접입력화면.png
   → 캡처: 08_손글씨입력버튼.png
   → 캡처: 09_사진업로드버튼.png

7. 프롬프트 입력 후 "만들기" 클릭
   → 캡처: 10_이미지생성중.png ⭐

8. 이미지 생성 완료
   → 캡처: 11_생성결과.png ⭐⭐⭐
   → 캡처: 12_다운로드공유버튼.png

9. "공유하기" 클릭 (선택사항)
   → 캡처: 13_QR코드모달.png

10. 상단 메뉴 → "내 작품 보기" 클릭
    → 캡처: 14_내작품목록.png

11. 작품 클릭
    → 캡처: 15_작품상세.png

⚠️ 중요: 생성된 이미지 2-3개 다운로드 저장!

───────────────────────────────────────────────────────────────
✍️ 4단계: 모듈 2 - 글쓰기 (15-20분)
───────────────────────────────────────────────────────────────

1. 홈 화면 → "글쓰기" 클릭
   → 캡처: 01_홈화면.png
   → 캡처: 02_글쓰기옵션.png

2. "연습하기" 클릭
   → 캡처: 03_연습하기화면.png

3. 뒤로 가기 → "장르 선택하기" 클릭
   → 캡처: 04_장르선택화면.png

4. "일기" 클릭
   → 캡처: 05_일기편집기_빈화면.png

5. 제목과 본문 입력
   예: 제목 "행복한 하루"
       본문 "오늘은 정말 행복한 하루였다..."
   → 캡처: 06_텍스트입력중.png

6. "AI 도우미" 버튼 클릭
   → 캡처: 07_AI도우미메뉴.png ⭐

7. "이어쓰기" 클릭
   → 캡처: 08_이어쓰기실행중.png (선택사항)
   → 캡처: 09_이어쓰기결과.png

8. "문법 교정" 클릭
   → 캡처: 10_문법교정결과.png

9. 글 완성
   → 캡처: 11_완성된글.png ⭐⭐⭐

10. "이미지 생성" 클릭 (선택사항)
    → 캡처: 12_이미지생성.png

11. "저장" 클릭
    → 캡처: 13_저장완료.png

12. "내 작품 보기" 클릭
    → 캡처: 14_내작품목록_글.png

13. 작품 클릭
    → 캡처: 15_작품상세_글.png

⚠️ 중요: 완성된 글 TXT 또는 DOC 파일로 저장!

───────────────────────────────────────────────────────────────
📚 5단계: 모듈 3 - 동화책 만들기 (20-30분) ⭐⭐⭐
───────────────────────────────────────────────────────────────

1. 홈 화면 → "동화책 만들기" 클릭
   → 캡처: 01_홈화면.png
   → 캡처: 02_동화책시작화면.png

2. 주제 입력창 클릭
   → 캡처: 03_주제입력화면.png

3. 주제 입력: "우주를 여행하는 용감한 토끼 이야기"
   → 캡처: 04_주제입력완료.png

4. 스타일 선택 (있는 경우)
   → 캡처: 05_스타일선택.png

5. "만들기" 클릭
   → 캡처: 06_AI초안생성중.png ⭐

6. 3페이지 초안 생성 완료
   → 캡처: 07_초안완성_3페이지.png ⭐⭐⭐

7. "편집하기" 클릭
   → 캡처: 08_동화책편집기.png

8. 페이지 텍스트 수정
   → 캡처: 09_텍스트수정.png

9. 각 페이지 "삽화 만들기" 버튼 표시
   → 캡처: 10_삽화생성버튼.png

10. "삽화 만들기" 클릭
    → 캡처: 11_삽화생성중.png ⭐

11. 삽화 생성 완료
    → 캡처: 12_생성된삽화.png ⭐⭐⭐

12. 전체 미리보기 (있는 경우)
    → 캡처: 13_전체미리보기.png

13. "PDF 내보내기" 클릭
    → 캡처: 14_PDF설정모달.png ⭐

14. PDF 생성 중
    → 캡처: 15_PDF생성중.png

15. PDF 다운로드
    → 캡처: 16_PDF다운로드.png ⭐⭐⭐

16. PDF 다운로드 실행!
    ⚠️ 필수: 세로 방향 1개 + 가로 방향 1개 = 총 2개 PDF
    
    첫 번째: 세로 방향 PDF
      - PDF 설정에서 "Portrait (세로)" 선택
      - 다운로드: 샘플_동화책_세로.pdf
    
    두 번째: 가로 방향 PDF
      - 다시 "PDF 내보내기" 클릭
      - PDF 설정에서 "Landscape (가로)" 선택
      - 다운로드: 샘플_동화책_가로.pdf

───────────────────────────────────────────────────────────────
💡 팁
───────────────────────────────────────────────────────────────

1. 화면 캡처는 순서대로!
   - 사용자 흐름을 따라가면서 캡처
   - 번호 순서가 기능 흐름과 일치

2. 로딩 화면도 꼭 캡처!
   - "AI가 만들고 있어요..." 화면
   - 프로그램 동작을 증명하는 중요 화면

3. 결과 화면이 가장 중요!
   - 생성된 이미지
   - 완성된 글
   - PDF 다운로드 화면

4. 실제 결과물 저장!
   - 이미지 파일 2-3개
   - 텍스트 파일 1-2개
   - PDF 파일 2개 (세로 + 가로) ⭐⭐⭐

5. 고해상도 유지!
   - 1920x1080 이상
   - PNG 형식 (JPG보다 선명)
